- `React` 18.2.0
- `Redux` 9.1.0
- `React Router` 6.11.0
- `styled-components` 6.1.8
- `Supabase` 2.42.0
- `Tailwind` 3.3.2

## Setting up locally ⚙️

### 1. Clone on your local machine

```bash
https://github.com/Jay-Wong-Git/wong-pizza.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run on your local server

```bash
npm run dev
```
